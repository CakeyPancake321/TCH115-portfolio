02 — Academics

Degree Objectives

Why I Chose This

I've been fascinated by computers since I was about three years old — how they work, how they communicate, how they can be taken apart and rebuilt. That curiosity never went away, and a degree in Network Security let me turn it into a career path.

Skills to Gain
  • →The knowledge to navigate any network confidently, regardless of its size or design
  • →Fast & reliable troubleshooting skills
  • →Fast & reliable security response skills
Career Connection

This degree is a direct path toward becoming a penetration tester / ethical hacker, while also building the broader skill set — network design, security tooling, scripting, policy, threat research — that supports SOC analyst and network technician roles along the way.

Network Infrastructure Design

Create a communications document for a network design, including hardware components, external connections, physical layer connectivity/addressing, and built-in operational and security components.

Security Tools Implementation

Install, configure, and test security hardware/software — port scanners, vulnerability and intrusion detection systems, firewalls, system hardening, anti-virus, patch management, auditing — with supporting documentation.

Automation & Scripting

Construct, implement, and document a script or program that automates a security-related process — installation, administration, patch management, auditing, analysis, etc.

Policy & Procedure Development

Create a policy or procedure such as a disaster recovery plan, business continuity plan, incident response policy, acceptable use document, or security policy.

Legal & Ethical Research

Develop a research report or implementation plan on legal/ethical best practices and mandated requirements in information security.

Threat Research & Evaluation

Research, document, test, and evaluate current industry threats, risks, malicious activity, covert methodology, encryption, and mitigation techniques to protect data confidentiality, integrity, and availability.
